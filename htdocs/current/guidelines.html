Guidelines and Procedures for Establishing and Maintaining EPD Cross-References

Purpose of this document:

The purpose of this document is to make clear for each type of Cross-references:
  1. The type-specific format of the cross-reference.
  2. The biological meaning of the cross-reference.
  3. The conditions and rules for establishing and maintaining of a cross-reference.
  4. The strategies for identifying new targets.
  5. The procedures for storage, update, verification of existing cross-references.

DR EPD

1. Format: 
    DR   EPD; "AC"; "ID"; "definition"; "position; direction"
    "AC"  string  EPD accession number.
    "ID"  string  EPD identification. 
    "definition"  keyword alternative promoter
    "position; direction"  seq_pos  Position and direction relative to the transcription initiation site.
2. Biological meaning:
    Multiple initiation sites preceding the same structural gene appear as alternative promoters if they are clearly separated from each other or differentially regulated.


3. Conditions and rules:

    The minimum distance required between two alternative initiation sites is 20 bp.

DR TRANSFAC

1. Format: 
    DR   TRANSFAC; "AC"; "ID"; "position"; "type".
    "AC"  string  TransFac accession number.
    "ID"  string  Transfac identification. 
    "position"  seq_pos  Position of site relative to the EPD initiation site.
    "type"  keyword  Type of initiation sites. 

    Notes:
    seq_pos is a positional sequence object represented according to
    the Milano convention.

    Valid keywords for "type" are: "by position", "by function".

2. Biological meaning:
    There are two types of cross-references, distuinguished by the keywords "by position" and "by function". "By position" means that the site described in TRANSFAC falls withing the range [-500,+100] of the initiation sites. "By function" means that the site has been reported to regulate transcription from the initiation site described by the EPD entry. "By position" cross-references are typically generated by an automatic procedure. "By function" cross-references are established by literature scanning. "By position" generally does not exclude "by function"; it simply means that a functional relationship has not (yet) come to the attention of the EPD curator.


3. Conditions and rules:

    Rule 1: Only site entries describing inidividual sites (as opposed to consensus sequences) which contain at least one SQ line, and share at least one EMBL cross-reference with a corresponding EPD entry can be cross-referenced cross-referenced.

    Rule 2: For a given pair of a TRANSFAC and an EPD entry, there can be at most one cross-reference for each SQ line in the TRANSFAC entry. A cross-reference can only be created for an SQ line, if the sequence matches a range of an EMBL sequence indicated


4. Target identification:

    Difficult TRANSFAC entries:

    TRANSFAC entries containing multiple SQ lines with identical sequences may cause promblems to automatic procedures for establishing "by position" cross-references. There is a program for identification of such entries which can be used in the following way: 

       g77 -fno-automatic /home/epd/src/find_difficult_sites.f 
   ./a.out < /db/transfac/site.dat

APPENDICES

A1 Representation of Positional Sequence Objects Convention:

1. The Milano convention

The Milano convention for defining the position of SEQUENCES relative to a transcription inititation SITE (or another physiological sites wich logically falls between adjacent bases) is ilustrated by the following example.
    EPD EMBL and TRANSFAC cross-references: 
               -15       -10        -5         0         5               
     ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' '       
      a C C C G C C t g c a c c c g a t t c a t g t g a g a a        
       <----------->                       |                         
       TRANSFAC entry R99999/ET$SITE_1     +---> EPD entry EP99999   
                                                                     
     <------------------------------------------------------->       
     EMBL entry ZZ999999.1/HS28BP                                    




                                                                         
     DR   EMBL; ZZ999999.1; HS28BP; [-19, 9].                        
     DR   TRANSFAC; R99999; ET$SITE_1; [-18,-12], by position.       
TRANSFAC applies the EMBL feature table numbering system for indicating a SITE or SUBSEQUENCE within a SEQUENCE. (Note that this problem is slightly different from the problem of indicating the position of a SEQUENCE relative to a SITE.) The FP lines of EPD follow the EMBL convention by referring to the first base of the transcript as the transcription initiation site.
    TRANSFAC-EMBL cross-references and EPD FP lines: 
         1       5        10        15        20        25               
     ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' ' '         
     a C C C G C C t g c a c c c g a t t c A T G T G A G A A         
       <--------->                         |                         
      TRANSFAC entry R99999/ET$SITE_1      EPD entry EP99999         
                                                                     
     <----------------------------------------------------->        
     EMBL entry ZZ999999.1/HS28BP                                     




    
     AC   R99999;                                                     
     ID   ET$SITE_1;                                                   
     DR   EMBL: ZZ999999; HS28BP(2:7)                                   
                                                                      
     FP   ET strange promoter :-S  ETL:R99999.1   1+      20; 99999.   

2. Format for sequence object positions

    ["begin","end"]
    where "begin" and "end" are integers reflecting the start and end position of the biological sequence object, respectively.
